This page is about current development status in regard to the mod supported versions of **Minecraft** and **Create**.

As being the only developer on **C:Metallurgy**, I cannot give out any ETAs 📆 for future releases given the progress on this project depends on many unpredictable factors. Furthermore, giving deadlines results in a lot of pressure and becomes counterproductive.
  
📣 Stay tuned to the latest announcements directly in the [C:Metallurgy Discord](https://discord.gg/KGQr5a2qc9).
