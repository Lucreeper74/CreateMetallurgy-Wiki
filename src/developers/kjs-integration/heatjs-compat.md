Alongside the native **KubeJS** :kjs: integration, **C: Metallurgy** also offers a [HeatJS](https://www.curseforge.com/minecraft/mc-mods/create-heat-js) 🔥 compatibility.

> [!IMPORTANT]
> In order to use the native **Create HeatJS** integration, you must install the [Create HeatJS](https://www.curseforge.com/minecraft/mc-mods/create-heat-js).

All recipes except Crucible Recipes are compatible with **HeatJS**🔥 using `.heatLevel("YOUR HEAT ID")` method.

::: tip NOTE
For further informations about [HeatJS](https://www.curseforge.com/minecraft/mc-mods/create-heat-js) 🔥 custom heat registering script, see [HeatJS Github README](https://github.com/XiaoHuNao/CreateHeatJS).
:::
