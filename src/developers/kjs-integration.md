**C: Metallurgy** offer an integration with **KubeJS** :kjs: natively which is very useful for packdevs.


> [!IMPORTANT]
> Since [v1.0.2](https://modrinth.com/mod/create-metallurgy/version/1.0.2-1.21.1), to use the native **KubeJS** :kjs: integration, you must install the [KubeJS](https://modrinth.com/mod/kubejs) mod **AND** [KubeJS Create](https://modrinth.com/mod/kubejs-create/versions). You **NEED BOTH** in order to use the **KubeJS** :kjs: integration.

::: warning
A minimum knowledge of **KubeJS** and JavaScript is needed, please refer to the [KubeJS Wiki Page](https://kubejs.com/wiki)
:::
